void Array_capacitySet (  Array *    array,
int    capacity
) 
 

Make the array a specified size: if the size is larger than the current size, an array of the specified size is created and the contents of the existing array are copied in.

This routine should be used to shrink the required memory if a large array is used temporarily. This is not done automatically by Array_empty( ) or Array_remove( ).

Parameters:
array Array*
capacity new capacity
