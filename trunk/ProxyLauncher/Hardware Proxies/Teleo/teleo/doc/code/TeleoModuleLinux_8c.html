Detailed Description

Sets up the Teleo SDK under Linux.

Illustrates the connection of the lower layers of the SDK for Linux. Used in several example apps. See TeleoModuleTest.c

Sets up a Channel Manager and a Module Manager. Reports modules, incoming messages, button presses, etc. Also runs threads for timer and input.

Uses pthreads.
