Detailed Description

Module Manager API.

Contains the structures, definitions and functions for the Teleo Module Manager. The Module Manager is responsible for receiving raw messages from the Teleo Channel Manager and interpretting them as messages from Teleo Modules.

cchar *    address,
cchar *    type,
bool *    bound,
TMM_Module **    tmmm
) 
 

Find or Create a module.

If a matching module already exists, the existing pointer is returned. If no module exists and the supplied spec (address and type) are complete, a module will be created.

At create time, the caller's data structures will not yet be set up properly. To avoid race-condition nastiness, the bound callback is not invoked in the case of existing modules, instead the bound flag is set. The caller then should invoke whatever bound( ) callback function is required.

Parameters:
tmm TeleoModuleManager
address the module address
type the module type
bound pointer to bound variable
tmmm return variable for new Module
Returns:
TELEO_OK (0) or appropriate error code
